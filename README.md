# Graph Visualizer

Interactive graph algorithm visualizer for creating, editing, and animating graph traversals. Part of the USACO Guide website.

## Features

- **Interactive Graph Canvas**: Create and edit graphs with drag-and-drop nodes and edges
- **Algorithm Animation**: Step-by-step visualization of graph algorithms (BFS, DFS, etc.)
- **Multiple Layouts**: Force-directed, tree, and circular layouts
- **Video Export**: Export animations as MP4 videos
- **Script Tracing**: Execute custom JavaScript to generate algorithmic steps

## Development

```bash
npm install
npm run dev
```
